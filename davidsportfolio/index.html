About

I am a full stack web developer from Lewiston, Idaho and I have my Web Development Bacholers degree from Lewis-Clark State College. I am learning something new about web development every chance I get, and enjoy keeping up to date with the web development world.

For backend technologies I mainly use and know PHP and MySQL. I have made several web applications with PHP and have professional experience adding extra functionality to WordPress sites with PHP. I understand modern PHP concepts including object oriented programing, input filtering, MVC program organization, and PDO objects.

Some other server side languages I have had experience with are C# with ASP.net MVC 5, and Python.

Front-end development is my strongest skill set. I have a lot of experience using HTML5, CSS3, JavaScript, JQuery, Sass, and web frameworks like Foundation and Bootstrap. Aside from the standard JQuery library I have also used JQuery mobile and JQuery UI for development. I also have experience using Adobe Photoshop and Adobe Illustrator to make and edit web graphics and pictures.
